# Консольное приложение для сбора статистики
 
*Инструкция*

1 Авторизоваться в teleport 
2 Выполнить команду "tsh proxy db <Имя кластера> --db-user adviser --db-name <Имя системной таблицы>"
2 В классе Variables заполнить соотвествующие поля:
	- SqlStatistic - sql запрос по сбору статистики
	- SystemDBName - название системной БД
	- SSLRootCert - корневой SSL сертификат
	- SSLCert - SSL сертификат
	- SSLKey - приватный ключ
3 Запустить приложение
4 В результате выполнения в месте запуска приложения будет сформирован файл со статистикой "statistic.xlsx"
